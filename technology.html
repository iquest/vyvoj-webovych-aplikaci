Technologie

Technologie hrají významnou roli v procesu vývoje webových aplikací. Mít základní přehled o technologiích je výhodné zejména proto, že dostanete lepší představu o technologických možnostech vašeho projektu.

Technologie pro frontend

HTML a kaskádové styly CSS

Na základě předchozí diskuze vyplynou nějaké požadavky na funkce a informace, které by měla mobilní aplikace obsahovat. Vytvoří se tedy návrh zadání, který se opět diskutuje se zákazníkem.

JavaScript (JS)

Je skriptovací jazyk, který je používaný především u webů a webových aplikací. Používá se k vylepšení HTML stránek a běžně se nachází v HTML kódu. JavaScript tvoří interaktivní a dynamický obsah webu. JS umožňuje reagovat na události, ukazovat speciální efekty, ověřovat data, vytvářet soubory cookies atd.

Technologie pro backend

Ruby (Ruby on Rails)

je framework pro vytváření webových aplikací. Je založen na nejmodernějších technologiích, které umožňují snadno a rychle vytvářet webové aplikace na míru zákazníka. Díky jasně definovaným konvencím, jako např. MVC (Model-View-Controller), lze za velice krátký čas postavit funkční základ aplikace v době, kdy jiné frameworky teprve začínají s konfigurací systému.

PHP (Laravel)

je framework pro vytváření webových aplikací v jazyce PHP podle architektonického vzoru MVC.

Python (Django)

je framework pro vytváření webových aplikací napsaných v Pythonu. Zaměřuje se na znovupoužitelnost a propojitelnost komponent a rychlý vývoj.

Java (Spring)

jedná se o framework, který poskytuje podporu infrastruktury pro vývoj Java aplikací. Spring je jeden z nejpopulárnějších rámců Java EE a pomáhá vývojářům vytvářet vysoce výkonné aplikace.
